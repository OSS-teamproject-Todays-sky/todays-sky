기상청 단기예보 API 날씨 정보 조회 스크립트

📝 개요 (Overview)
이 파이썬 스크립트는 기상청 단기예보 조회서비스 API를 활용하여 특정 지역의 날씨 데이터를 가져옵니다. API로부터 받은 원본(Raw) 데이터를 팀 프로젝트에서 사용하기 쉬운 간결한 JSON 형식으로 가공하여 출력하는 것을 목표로 합니다.

현재 코드는 부산광역시 남구 대연동을 기준으로 설정되어 있습니다.

✨ 주요 기능 (Features)
기상청 단기예보 API를 호출하여 날씨 데이터를 요청

API의 복잡한 데이터 발표 시간을 자동으로 계산하여 가장 최신 정보 조회

수신한 원본 데이터를 '현재 날씨' 중심으로 가공

현재 기온, 최고/최저 기온, 하늘 상태, 강수 정보 등 핵심 데이터를 추출

최종 결과를 다른 서비스(백엔드, 프론트엔드)에서 사용하기 용이한 JSON 형식으로 출력

🛠️ 요구사항 (Requirements)
Python 3.x

⚙️ 설치 및 설정 (Installation & Setup)
1. 필요 라이브러리 설치 - 터미널에서 아래 명령어를 실행하여 requests 라이브러리를 설치합니다.
# pip install requests

2. API 인증키 설정 - weather.py 파일을 열어 아래 부분에 공공데이터포털에서 발급받은 본인의 API 인증키를 입력합니다.
# API_KEY = "1o00nZt0TaGNNJ2bdH2hZw"

3. (선택) 위치 좌표 변경 - 다른 지역의 날씨를 조회하고 싶다면, 아래 NX, NY 값을 원하는 지역의 격자 좌표로 변경합니다. (좌표는 기상청 격자좌표 변환표 참고)
# 위치 좌표 (현재: 부산 남구 대연동): NX = 98 / NY = 75
